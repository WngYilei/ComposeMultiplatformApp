# ComposeMultiplatformApp
这是一个学习Compose跨平台的项目，以一个wanandroid  api 完成的简单项目，    主要是Android端和ios端。
技术栈：

- kotlin 1.8.0

- Compose 1.3.0

- Ktor 2.2.2
